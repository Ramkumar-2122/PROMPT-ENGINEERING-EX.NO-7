# Ex.No.7 – Development of a Prompt-Based Application Using Large Language Models

## Date: 26-05-26
## Register No: 21222322085

---
# Aim

To develop a prompt-based application using ChatGPT and demonstrate how prompts can be used to create a personal productivity assistant that manages daily tasks, schedules reminders, provides wellness tips, and answers general user queries using natural language interaction.

---

# AI Tools Required

* ChatGPT
* Large Language Models (LLMs)
* Python (Optional)
* Web Browser
* Prompt Engineering Techniques

---

# Introduction

Large Language Models (LLMs) have transformed the way humans interact with Artificial Intelligence systems. Prompt-based applications use carefully designed prompts to guide AI models in performing useful real-world tasks.

This experiment focuses on designing a Personal Productivity Assistant powered by ChatGPT. The assistant helps users manage tasks, schedule reminders, organize activities, provide wellness suggestions, and answer general questions through natural language communication.

The experiment demonstrates how prompt engineering can be used to create intelligent and adaptive systems that improve productivity and support everyday activities.

---

# Objective

1. To develop a prompt-based productivity assistant using ChatGPT.
2. To understand how prompts guide AI behavior.
3. To simulate natural language interaction.
4. To design prompts for different productivity-related tasks.
5. To study adaptability and personalization in AI systems.

---

# Problem Statement

Design a personal productivity assistant that can:

* Manage daily tasks
* Schedule reminders
* Suggest wellness tips
* Answer general queries
* Adapt to user preferences over time

The assistant should communicate naturally and provide useful responses for productivity improvement.

---

# Prompt Used

## Main Prompt

“Design a personal productivity assistant that can help manage daily tasks, schedule reminders, suggest wellness tips, and answer general queries. The assistant should interact using natural language and be adaptable to the user’s changing preferences over time.”

---

# Procedure

1. Define the core requirements of the productivity assistant.
2. Construct prompts for different functionalities.
3. Use ChatGPT to simulate user interaction.
4. Test prompts for task management and scheduling.
5. Generate wellness recommendations.
6. Simulate adaptive behavior based on user preferences.
7. Analyze outputs generated by the LLM.

# System Features

---

# 1. Daily Task Manager

## Prompt Example

“Remind me to submit my assignment tomorrow at 10 AM.”

---

## AI Response

“Reminder set successfully. You will be reminded tomorrow at 10:00 AM to submit your assignment.”

---

## Features

* Accepts natural language commands
* Organizes tasks by date and priority
* Tracks pending tasks
* Generates daily summaries

---

# 2. Smart Scheduler

## Prompt Example

“Schedule a meeting with my project team on Friday at 4 PM.”

---

## AI Response

“Your project meeting has been scheduled for Friday at 4:00 PM.”

---

## Features

* Creates schedules automatically
* Detects time conflicts
* Suggests free time slots
* Manages reminders

---

# 3. Wellness Tips Generator

## Prompt Example

“Give me a wellness tip for today.”

---

## AI Response

“Remember to take short breaks every hour to reduce eye strain and improve focus.”

---

## Features

* Provides health and wellness suggestions
* Encourages hydration and exercise
* Suggests screen-time breaks
* Adapts recommendations based on user behavior

---

# 4. General Query Assistant

## Prompt Example

“What is Artificial Intelligence?”

---

## AI Response

“Artificial Intelligence is a field of computer science that enables machines to perform tasks that normally require human intelligence.”

---

## Features

* Answers general questions
* Provides educational explanations
* Supports natural conversation

---

# 5. Preference Adaptation

## Prompt Example

“I prefer reminders in the evening instead of the morning.”

---

## AI Response

“Preference updated. Future reminders will be scheduled during evening hours whenever possible.”

---

## Features

* Learns user preferences
* Adapts interaction style
* Personalizes responses
* Simulates memory-based interaction
